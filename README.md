# My-Chat-Service

### ✨ Node.js , Koa, MongoDB , Pug , tailwind 사용

### 😊 Node.js/Koa 웹소켓을 활용한 실시간 채팅   
> 웹소켓을 활용한 실시간 채팅 서비스   
> 서버에 들어와 처음 메세지를 보낼때 닉네임 생성  
> 사용자의 닉네임은 이름은 미리 설정된 값을 조합하여 랜덤으로 생성  
> 채팅 기록은 MongoDB에 저장,  
> 서버에 재접속시 기존 채팅 내역이 출력   
  
  
  <br/>

```
src/config 폴더에 dev.js (MongoURL) 생성
npm run server
```
